## 存储结构
### 顺序存储（数组）
- O(1)随机访问
- O(n)插入/删除
### 链式存储（链表）
- 不支持随机访问
- O(1)插入/删除
- O(n)访问某个元素
- 比数组占用更多的存储空间
#### 不同的形式
- 单向链表
- 双向链表
- 循环链表
- 镜头链表
